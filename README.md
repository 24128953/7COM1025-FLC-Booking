# FLC Booking App

This repository contains the 7COM1025 coursework implementation for the Furzefield Leisure Centre booking system.

## Tech Stack

- Java 17
- Maven 3
- JUnit 5

## Project Structure

- `src/main/java` - application source code
- `src/test/java` - JUnit tests
- `pom.xml` - Maven build configuration

## Run

```bash
mvn compile
mvn test
```

At this stage, the repository contains a scaffold and package structure for incremental implementation.
