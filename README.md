# FLC Booking App

Scaffold project structure for the Furzefield Leisure Centre booking system.

## Current Structure

- `src/flcbookingapp/` - empty Java class shells
- `test/` - placeholder for JUnit tests
- `nbproject/` - NetBeans project metadata placeholder
- `build/` - build output directory placeholder
- `build.xml` - Ant build scaffold
- `manifest.mf` - application manifest scaffold
